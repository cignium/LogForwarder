Log Forwarder

Forward your logs to Splunk.

Use

Variables

Configure your extension setting up these values as application settings in your web app.

  • Log File Path (Relative to website's root)

    LOG_FILE_PATH

  • Set "1" to start LOG_FILE_PATH on Home (Optional, default 0)

    START_ON_HOME

  • Splunk's Index

    INDEX_NAME

  • Splunk's Hostname

    SPLUNK_HOSTNAME

  • Splunk's UserName

    SPLUNK_USERNAME

  • Splunk's Password

    SPLUNK_PASSWORD

  • Monitor Delay in seconds (Optional, default 5)

    DELAY
